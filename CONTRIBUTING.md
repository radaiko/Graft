# Contributing to Graft

Thank you for your interest in contributing to Graft!

## Getting Started

1. Fork the repository
2. Clone your fork
3. Create a feature branch from `main`

## Building

```bash
dotnet build src/Graft.sln
```

## Running Tests

```bash
dotnet test src/Graft.sln
```

## Project Structure

- `src/` — Source code and solution file
- `tests/` — Test projects
- `docs/` — Documentation (published via GitHub Pages)

## Pull Requests

- Keep PRs focused on a single change
- Include tests for new functionality
- Ensure all tests pass before submitting
